Important: Your redirect URI must match exactly in Google Cloud Console: {{.BaseURL}}/oauth/callback Copy this exactly - even http vs https or trailing slashes matter!
Step-by-step Google Cloud setup
  1. Go to Google Cloud Console and create or select a project
  2. Enable the Google Calendar API
  3. Configure the OAuth consent screen:
    • User Type: External
    • Fill in required fields (App name, User support email, Developer email)
    • On the "Scopes" page, click Add or Remove Scopes, search for calendar.events, check the box, and click Update
  4. Add yourself as a test user (required while app is in "Testing" status):
    • On the "Test users" page, click + ADD USERS
    • Enter your Google email address (the one you'll use to connect)
    • Click Add, then Save and Continue
  5. Go to Credentials → Create Credentials → OAuth client ID
  6. Application type: Web application
  7. Under "Authorized redirect URIs", click + ADD URI and paste:
    {{.BaseURL}}/oauth/callback
  8. Click Create, then copy the Client ID and Client Secret below
